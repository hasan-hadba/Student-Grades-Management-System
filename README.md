# Student Grades Management System 🎓

A professional C programming project designed to streamline the management of academic records. This tool was developed to demonstrate core computer science concepts learned through independent research and CS50 materials.

## 🚀 Key Features
- **Custom Data Structures:** Implements `structs` to organize student names, IDs, and grades.
- **Dynamic GPA Calculation:** Uses a specialized algorithm to process multiple subjects.
- **Performance Filtering:** Automatically categorizes students into:
  - 🌟 **Honor Roll** (GPA >= 90)
  - ✅ **Passing** (GPA >= 60)
  - ⚠️ **Academic Warning** (GPA < 60)

### 🛠️ Operation Steps:
1. **Running the Program:** Compiling and executing the `student.c` file.
2. **Selecting Number of Students:** Determining the size of the record array.
3. **Entering Student Data:** Inputting names, IDs, and subject grades.
4. **Final Results:** Generating the performance report with automated GPA calculation.

## 💻 Concepts Applied
- **Arrays & Loops:** For efficient data handling.
- **Modular Programming:** Clean code structure using custom functions.
- **User Interface:** Interactive CLI (Command Line Interface) for data entry.

## 🛠️ How to Run
1. Open a terminal in the project directory.
2. Compile using: `make student`
3. Run the program: `./student`
